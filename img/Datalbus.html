    Atualizar Firmware

    Atenção!
    Permitido apenas a atualização para a versão mais recente de firmware!
    Caso tenha necessidade de atualizar para uma versão diferente, entre em contato com o squad responsável por gerenciar firmwares

    Calibrar Inercial

    Atenção!
    Este comando irá iniciar o processo de calibração do sensor inercial, antes confira os seguintes passos:
    • Unidade está comunicando?
    • Unidade está instalada corretamente?
    • Unidade está com a última versão de firmware?
    • Em caso de recalibração, checar histórico de eventos deste carro para ter certeza que há um erro.
    A calibração só será finalizada quando o veículo se deslocar por uma distância mínima garantindo que o equipamento se calibre corretamente, portanto este processo poderá demorar.
    Não há um tempo ou distância mínima para que isso ocorra.
    Os futuros eventos relacionados ao inercial poderão ter uma quantidade diferente quando comparado ao eventos anteriores da data do agendamento do comando.
